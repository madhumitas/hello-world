Whole Slide Image Repositories Analysis

The WSI are difficult to view and annotate as they are extremely large to open and view using the common image viewing softwares. In order to view these images we need special applcations. These applciations work by reading these images and then splitting itinto tiles and these tiles are then served to one of the web application which makes use of the viewers like WebGL or OpenLayers.
